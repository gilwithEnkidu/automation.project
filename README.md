## automation project
데이터의 취합, 가공, 보고서 발송에 이르는 파이프라인 구축 시뮬레이션  



## 구조
  + 'raw.data'    측정장비에서 출력한 데이터 모음
  + 'processed'   규격 판단에 따른 가공 데이터 모음
  + 'reports' 발송된 리포트 모음
  + 'scripts' 사용된 파이썬 코드 모음
